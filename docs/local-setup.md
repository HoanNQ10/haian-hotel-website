# Local Setup

## Prerequisites
- Node.js (version TBD)
- npm or yarn (TBD)
- Git

## Installation Steps

### 1. Clone the Repository
```bash
git clone <repository-url>
cd D:\Projects-hotel-website
```

### 2. Install Dependencies
```bash
npm install
# or
yarn install
```

### 3. Environment Setup
- Copy `.env.example` to `.env.local` (if applicable)
- Configure environment variables (TBD)

### 4. Development Server
```bash
npm run dev
# or
yarn dev
```

### 5. Build for Production
```bash
npm run build
# or
yarn build
```

### 6. Preview Production Build
```bash
npm run preview
# or
yarn preview
```

## Development Guidelines
- Use the documentation in `docs/` folder as reference
- Follow the requirements in `docs/requirements.md`
- Use content from `docs/content.md`
- Follow design guidelines in `docs/design.md`

## File Structure
- `src/`: Source code
- `public/`: Static assets
- `docs/`: Documentation
