# Local Setup & Development Guide

## 📋 Yêu Cầu Hệ Thống

- **Node.js** >= 18.x
- **npm** >= 9.x (hoặc yarn, pnpm)
- **Git** (để quản lý version)

## 🚀 Cài Đặt & Chạy Locally

### 1. Cài Đặt Dependencies

```bash
npm install
```

### 2. Chạy Development Server

```bash
npm run dev
```

Server sẽ khởi động tại **http://localhost:4321/**

Mở URL này trong trình duyệt để xem website (auto reload khi edit).

### 3. Build cho Production

```bash
npm run build
```

Output được lưu trong folder `dist/` (HTML + CSS được minify)

### 4. Preview Production Build

```bash
npm run preview
```

Kiểm tra website đã build tại http://localhost:4321/

## 📱 Test trên Mobile

### Cách 1: Chrome DevTools (Recommended)

1. Mở http://localhost:4321 trong Chrome
2. Nhấn **F12** hoặc **Ctrl+Shift+I** để mở DevTools
3. Nhấn **Ctrl+Shift+M** để kích hoạt Mobile View
4. Chọn **iPhone 14 Pro** hoặc thiết bị khác từ dropdown
5. Test responsive layout & interactions

### Cách 2: Thiết Bị Thực

1. Tìm IP của máy tính (chạy `ipconfig` trên Windows hoặc `ifconfig` trên Mac)
2. Mở `http://<YOUR_IP>:4321` từ điện thoại (cùng mạng Wi-Fi)
3. Test tất cả tính năng, button clicks, scrolling

## ⚠️ Lưu Ý / Known Limitations

1. **Placeholder Images:** Tất cả ảnh hiện dùng placeholder.com. Cần thay bằng ảnh thực.
2. **System Fonts:** Dùng system fonts (sans-serif). Nên thêm Google Fonts (Playfair Display + Inter) sau.
3. **Button Links:** Các nút (Call, Zalo, Map) hiện là placeholder. Cần thêm URLs/phone numbers.
4. **Mobile Menu:** Hamburger menu chưa implement. Chỉ có hidden menu trên mobile.
5. **Contact Form:** Không có backend. Chỉ có static CTA buttons.
6. **SEO:** Basic meta tags có. Cần thêm Open Graph, canonical URLs, structured data.
7. **Analytics:** Chưa setup. Thêm Google Analytics/Plausible khi cần.
